1. Advancing Landmark Identification Through Cloud-powered Image Processing and Computer Vision: An In-Depth Analysis of "LANDMARKER"

  • Co-Author: Arif Najafov
  • Supervisor: Ph.D. Assoc. Prof. Leyla Muradkhanli
  • DOI: N/A yet, will be added later.
  • Certificate: View Certificate
Abstract:

This paper presents the development of a cloud-powered landmark recognition application, LandMarker, which utilizes Google Cloud Vision API and Streamlit for real-time image processing and geographical mapping. The application is designed to identify landmarks from user-uploaded images and provide detailed information about the landmark's history, architecture, and significance. The system is also integrated with TogetherAI API to generate AI-powered summaries of the landmark's key features. The paper discusses the technical aspects of the application, including the image processing techniques used for landmark recognition, the integration of AI-powered summaries, and the challenges faced during development. The results of the application's performance are presented, along with future directions for improving the system's accuracy and expanding its functionality.

Keywords:

Landmark Recognition, Image Processing, Cloud Computing, Google Cloud Vision API, Streamlit, TogetherAI API
